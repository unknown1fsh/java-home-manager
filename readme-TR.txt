Hoş geldiniz!

Java Home Manager kullanmayı tercih ettiğiniz için teşekkür ederiz. 
Bu kurulum sihirbazı, sisteminizde Java ortamlarını hızlıca kurmanıza ve yapılandırmanıza yardımcı olacaktır.

Kurulum için aşağıdaki adımları takip edin:

1. 'JavaHomeManagerSetup.exe' dosyasına çift tıklayın.
2. Windows size "Bu uygulamanın cihazınızda değişiklik yapmasına izin verilsin mi?" diye soracak. **Evet** seçeneğini tıklayın.
3. Kurulum sihirbazındaki adımları takip edin.
4. Kurulum sırasında Masaüstü kısayolu oluşturmak isteyip istemediğiniz sorulacaktır (önerilir).
5. Kurulum tamamlandıktan sonra Java Home Manager otomatik olarak çalışacaktır.

Java Home Manager Ne İşe Yarar:
- Bilgisayarınızda yüklü olan tüm Java sürümlerini listeler.
- İstediğiniz Java sürümünü kolayca seçmenizi sağlar.
- JAVA_HOME ve sistem PATH değişkenlerini otomatik olarak günceller.

Önemli Notlar:
- Kurulum sırasında yönetici (Administrator) yetkileri gereklidir, çünkü program sistem ortam değişkenlerini değiştirmektedir.
- Kurulumdan sonra Masaüstü veya Başlat Menüsü'ndeki kısayolu kullanarak Java Home Manager'ı istediğiniz zaman çalıştırabilirsiniz.
- Java Home Manager'ı dilediğiniz zaman "Programlar ve Özellikler" bölümünden kaldırabilirsiniz.
